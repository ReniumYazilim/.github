# VS Code Sik Kullanilan Kisayollar

## Favoriler

| Kisayol | Islem |
|---------|-------|
| `Ctrl+Shift+P` | Command Palette (tum komutlara erisim) |
| `Ctrl+P` | Dosya arama (Quick Open) |
| `Alt+Z` | Kelime bol / Word Wrap (uzun satirlari ekrana sigdir) |
| `Ctrl+,` | Ayarlar (Settings) |
| `` Ctrl+` `` | Terminal ac/kapat |
| `Ctrl+B` | Sidebar ac/kapat |
| `Ctrl+Alt+B` | Chat ac/kapat |
| `Ctrl+W` | Aktif sekmeyi kapat |
| `Ctrl+Shift+N` | Yeni pencere |

## Pencere ve Goruntuleme

| Kisayol | Islem |
|---------|-------|
| `Ctrl+Shift+P` → `Reload Window` | VS Code'u yeniden baslat (en hizli yol) |
| `Ctrl+Shift+P` → `Close Window` | Pencereyi tamamen kapat |
| `Alt+Z` | Kelime bol / Word Wrap (uzun satirlari ekrana sigdir) |
| `Ctrl+Shift+E` | Explorer panelini ac |
| `Ctrl+Shift+G` | Git panelini ac |
| `Ctrl+Shift+X` | Extensions panelini ac |
| `F11` | Tam ekran |
| `Ctrl+\` | Editoru yana bol (split) |
| `Ctrl+1/2/3` | Editor gruplari arasinda gecis |

## Duzenleme

| Kisayol | Islem |
|---------|-------|
| `Ctrl+D` | Secili kelimeyi bir sonraki eslesmede de sec (multi-cursor) |
| `Ctrl+Shift+L` | Secili kelimenin tum eslesmelerini sec |
| `Alt+Up/Down` | Satiri yukari/asagi tasi |
| `Shift+Alt+Up/Down` | Satiri kopyala (yukari/asagi) |
| `Ctrl+Shift+K` | Satiri sil |
| `Ctrl+L` | Satiri sec |
| `Ctrl+/` | Yorum ekle/kaldir |
| `Ctrl+H` | Bul ve degistir |
| `Ctrl+Shift+H` | Tum dosyalarda bul ve degistir |
| `Tab` | Girinti ekle |
| `Shift+Tab` | Girinti kaldir |

## Arama ve Navigasyon

| Kisayol | Islem |
|---------|-------|
| `Ctrl+F` | Dosya icinde ara |
| `Ctrl+Shift+F` | Tum dosyalarda ara |
| `Ctrl+G` | Satir numarasina git |
| `F12` | Tanimina git (Go to Definition) |
| `Alt+Left/Right` | Geri/ileri navigasyon |
| `Ctrl+Tab` | Acik sekmeler arasinda gecis |

## Multi-Cursor

| Kisayol | Islem |
|---------|-------|
| `Alt+Click` | Ek cursor ekle |
| `Ctrl+Alt+Up/Down` | Yukari/asagi cursor ekle |
| `Ctrl+D` | Sonraki eslesen kelimeyi sec ve cursor ekle |

## Terminal

| Kisayol | Islem |
|---------|-------|
| `` Ctrl+` `` | Terminal ac/kapat |
| `` Ctrl+Shift+` `` | Yeni terminal olustur |

## Debug

| Kisayol | Islem |
|---------|-------|
| `F5` | Debug baslat |
| `F9` | Breakpoint ekle/kaldir |
| `F10` | Step Over |
| `F11` | Step Into |
