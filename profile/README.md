# Renium Global

**No-Code ile Sınırsız Çözümler**

Renium Global, işletmelerin dijitalleşme yolculuğunda **OCTO** adlı yenilikçi no-code platformuyla hizmet veren bir teknoloji şirketidir. Kodlama bilgisi gerektirmeden, geleneksel ERP sistemlerine uygun maliyetli bir alternatif sunar.

---

## 🚀 OCTO Platform

İşletmenizin tüm operasyonel süreçlerini tek platformda yönetin:

| Modül | Modül |
|-------|-------|
| Satış Yönetimi | Satın Alma |
| Stok & Depo Yönetimi | Üretim Yönetimi |
| Kalite Yönetimi | Maliyet Yönetimi |
| Proje Yönetimi | Personel Yönetimi |
| Bakım Yönetimi | Cari Yönetim |

---

## 💼 Hizmetlerimiz

- **OCTO No-Code Platform** — Kod yazmadan iş süreçlerinizi dijitalleştirin
- **ERP Hizmetleri** — Uçtan uca iş süreçlerini tek platformda yönetin
- **Veri Analizi** — Verilerinizi stratejik iş zekasına dönüştürün
